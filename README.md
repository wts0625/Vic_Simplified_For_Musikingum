与完整VIC模型的模块差异
模块/功能	本代码实现	完整VIC（4.x/5.x）
能量平衡	简化，仅用气象数据估算PET	完整求解能量平衡（净辐射、感热、潜热、土壤热通量）
土壤温度	未模拟	模拟土壤温度及热传导
雪模块	无	有（积雪、融雪、雪水当量）
植被过程	仅冠层截留和统一蒸发，无气孔导度、蒸腾区分	多植被类型，包含气孔导度、蒸腾、光合作用（新版）
土壤分层	固定3层，厚度0.1/0.5/0.5 m	一般2层或3层，厚度可变，参数更丰富
产流机制	变入渗曲线 + ARNO基流，但参数固定	相同理论基础，但允许参数空间分布
基流平滑	额外增加了GW_store线性水库平滑	基流直接由公式计算，无额外平滑（除非汇流模块）
汇流	分布式等流时线（线性滞后）	通常需外接河道汇流模型（如Rout）或采用Muskingum
数据同化/优化	无	可能包含参数率定、数据同化接口
空间分布	分布式输入，参数空间均匀（或由数据提供）	完全分布式，参数可随网格变化
湖泊/湿地/灌溉	无	可能有相应模块
输出	子流域出口流量，以及汇总水量平衡	多种输出选项（径流、蒸发、土壤水、能量通量等）

使用了的vic模型机制包括：
使用的VIC机制
变下渗曲线（Variable Infiltration Curve）

采用基于ARNO/Xinanjiang模型的入渗容量曲线，根据土壤含水量计算地表径流和入渗，是VIC核心产流机制之一。

基流模块（ARNO型）

使用参数 D_S、D_SMAX、W_S 构建的非线性基流关系，模拟深层土壤排水。

土壤分层与水量平衡

将土壤分为三层，分别计算水分变化，并考虑层间下渗和溢出。

冠层截留与蒸发

降雨首先被冠层截留，再以PET速率蒸发，剩余PET用于土壤蒸发。

蒸发分配

剩余PET优先蒸发W1（表层），再蒸发W2（第二层），近似VIC的蒸发分层策略。
